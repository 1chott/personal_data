1、操作系统
    操作系统 ====》 协调者
    
    规章制度 类比  操作系统
    员工     类比  程序
    
    作用：向下封装硬件，向上提供接口
    
2、目录结构
    1）不同点
        a) windows以盘符作为管理单位
        b) linux以文件夹（目录）作为管理单位
    
    2）相同点
        都是以目录方式组织
        
    3）linux常用目录
    /: 根目录，所有目录从这开始
    /home: 用户所在的目录，如 /home/python，注意：以后的操作尽量在用户目录，在用户目录有权限做任何事情，在别的目录需要权限
    /bin:  放命令，工具
    
3、命令
    1) 为什么需要学习命令
        a) 操作更快
        b) 有些系统没有界面，只能通过命令操作
        
    2) 命令解析器
    命令解析器作用：把命令翻译一下，解析成计算机能识别的二进制
    
    命令解析器也是一个软件，这个软件自带一些命令，有可能这些命令和/bin中命令重名
    
    输入命令，优先从命令解析器自带中命令中找，找不到再从/bin，再找不到报错
    
    3）内建命令和外部命令
     命令解析器也是一个软件，这个软件自带一些命令：内建命令
     /bin中命令：外部命令
     
        python@ubuntu:~$ type -a cd         #查看cd的命令类型
        cd 是 shell 内建
        python@ubuntu:~$ type -a echo       #查看echo的命令类型
        echo 是 shell 内建
        echo 是 /bin/echo
    
    4) 命令格式和方法
        1、帮助文档
            a) 外部命令
                ls --help   #参看ls的使用方法
                man ls      #通过man手册查看ls的使用，按方向键移动，按q退出
    
            b) 内建命令
                help cd     #查看cd的使用方法
    
    
        2、快捷键
        a) clear: 清屏    ctrl+l(字母)
        b) ifc   按tab自动补全
        c) ctrl+c 中断操作
            ping 127.0.0.1  有信息后再按ctrl+c
    
         移动到头部： ctrl + a（Home）
       移动到尾部： ctlr + e（End）
    
    5) 目录切换
    a) 相对路径和绝对路径
        绝对路径：完整唯一的路径，如果是linux，绝对路径从/开始
        相对路径：相对于当前路径
        
        /home/python/workspace 绝对路径
        home/python/workspace  相对路径，当前路径下的home，home下的python ....
        
        cd	切换到当前用户的主目录(/home/用户目录)
        cd ~	切换到当前用户的主目录(/home/用户目录)
        cd .	切换到当前目录
        cd ..	切换到上级目录
        cd -	可进入上一个进入的目录
        
        cd /home   根目录下的home
        
        # 当前路径下的home
        cd ./home
        cd home    #和上面等价
        
        # 当前路径下的上一级
        cd ../home
        
     6) 显示文件信息
     ls             显示文件信息，不包括隐藏文件
     ls -a          显示所有的文件信息，linux下.开头的文件是隐藏文件
     ls -l          list, 以列表方式显示
     ls -lh(ls -l -h)   h配合l使用，大小单位自动转换，根据人性化显示
     ls -lah
     
     -rw-rw-r--  1 python python    0 4月  13 10:05 1.txt
     第一个字符：文件类型，- 普通文件  d 目录文件
     
    ls *.txt 查看所有后缀为.txt的文件
    ls *.*   查看中间带.的文件
    
    
    7) touch 
    如果文件不存在，新建，如果存在，把文件创建时间改为当前时间，只能新建普通文件，不能是文件夹
    
    
    8) tree
    sudo apt-get install tree
    
    以树状方式显示目录结构
    tree
    tree -L 2 只显示2层
    
    9) 重定向
    ls -l > xxx.txt
    ls -l本来显示在屏幕的内容，通过>重定向，内容不显示屏幕，显示到文件，文件不存在新建，存在覆盖
    
    
    ls -l >> xxx.txt  文件不存在新建，存在追加
    
    
    cat xxx.txt 显示xxx.txt的文件内容，打印在终端上
    
    10) ln 创建链接文件
    软链接：ln -s 源文件  软链接文件名
            ln -s xxx.txt soft.txt
    
    
        1) windows的快捷方式
        2) 打开链接文件，实际上打开的是源文件
              soft.txt                 xxx.txt
        3) 可以简单认为软链接文件不占内容，固定很小的大小
        4) 删除源文件(xxx.txt)，链接文件就失效(soft.txt)
        
    
    硬链接：ln 源文件  硬链接文件名         注意：只有普通文件可以创建硬链接
        python内存，云盘的秒传，C++11智能共享指针   和硬链接原理一样
        
         ln yyy.txt h1.txt
         
         1) 给yyy.txt文件内容起了2个名字，一个叫yyy.txt，一个叫h1.txt, 在硬盘中只有一份文件
         2) 删除一个，还有一个名字，只有把所有的文件名删除，硬盘才会真正删除文件内容
         3) 改变其中一个，另外一个会一样，因为用的是同一份文件内容
         
         硬链接防止误删文件
    
    
    11) mkdir
        新建文件夹，只能是文件夹，不能是普通文件
        mkdir mike 创建一个文件夹名字叫mike
        mkdir a/b/c -p   -p作用递归创建文件夹
